Pomodoro App

From the SlowTypists Team

This task management application allows an admin to maintain a database of users and their tasks. It also allows tasks to be labeled with multiple tags and assign users badges when they have completed certain goals. The admin can interact with the application in two ways:

By Entity

The admin can select a specific entity in the nav bar to view, create, update, or delete an entry in the database.

By User

The admin can search for a user to bring up their dashboard and to see that user's upcoming tasks, earned badges, and start a Pomodoro task timer for them.
